### 명령어 정리 01

#### 라우터 설정
* user> en
관리자 모드 실행

* config terminal
설정 시작

* show ip interface brief
포트번호, 인터페이스 등 확인 가능

* #
전역 설정 모드

* interface [interface] [port-num]

* ip address [ip] [subnet mask]

* no sh
끄지 않는다. 즉 킨다

* ctrl + c
관리자 모드로 즉시 이동

* wr
